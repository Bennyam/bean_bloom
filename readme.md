# ☕ Bean & Bloom — Landing Page

A modern, responsive landing page for a fictional coffee & dessert bar, designed to deliver a premium user experience across all devices.

---

## 🌐 Live Demo

👉 https://bean-and-bloom.netlify.app

---

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scrolling navigation
- Animated mobile menu (hamburger → close transition)
- Micro-interactions (hover effects, image scaling, button states)
- Clean and modern UI design
- Contact form with focus states
- Consistent design system (colors, spacing, typography)

---

## 🧩 Sections

- Hero section with call-to-action
- “Onze Troeven” (Features)
- About section (brand storytelling)
- Product showcase (grid layout)
- Customer testimonials
- Contact form
- Footer with navigation & contact info

---

## 🛠️ Tech Stack

- **React**
- **Vite**
- **Tailwind CSS**
- **JavaScript (ES6+)**

---

## 🎨 Design Approach

The goal of this project was to create a **premium, clean, and user-friendly experience**.

Key design principles:

- Simplicity and clarity
- Strong visual hierarchy
- Consistent spacing and typography
- Subtle animations and interactions
- Mobile-first thinking

---

## 📱 Responsiveness

The layout adapts seamlessly across devices:

- Desktop: multi-column layouts
- Tablet: balanced stacking
- Mobile: optimized vertical flow with interactive menu
